
# Heart Disease Analysis

This project analyzes heart disease data using Tableau.

Dataset was cleaned and prepared for visualization.

Total Records: 4494  
Total Columns: 18  

Visualizations include:
- Gender-wise Heart Disease
- Age-wise Heart Disease
- Diabetes and Stroke Analysis
- Smoking & Alcohol Impact
- Race-wise Analysis
- BMI & Risk Study

Tools Used:
- Tableau
- Excel
- GitHub
