📁 Step 1: Connect Data

  1. Open Tableau Desktop or go to Tableau Public online

  2. Connect to Text File — select heart_disease_clean.csv from the analysis/outputs/ folder

  3. Review schema — confirm all columns imported correctly, especially date/numeric types

  4. Create extract for faster performance with 1,500+ rows

🎨 Step 2: Build Charts

  1. Sheet 1 — Age Distribution: Drag AgeGroup to Columns, COUNT(PatientId) to Rows, HeartDisease to Color

  2. Sheet 2 — Cholesterol Bar: CholBin → Columns, Count → Rows, HeartDisease → Color

  3. Sheet 3 — BP Heatmap: BPBin → Rows, AgeGroup → Columns, HeartDisease rate → Color (use table calculation)

  4. Sheet 4 — Regional Map: Region → Detail, Heart Disease Rate → Color with a geographic map

🖥️ Step 3: Dashboard Layout

  1. New Dashboard — set size to 1400×900 (Desktop)

  2. Add KPI tiles at the top using text objects with calculated fields for Total Patients, Prevalence %, Avg Age

  3. Arrange sheets in a 2×3 grid layout — age, gender, cholesterol, BP, lifestyle, regional

  4. Add a filter action: Dashboard → Actions → Filter — link AgeGroup across all sheets

🌐 Step 4: Publish & Embed

  1. Save as .twbx (packaged workbook with embedded data)

  2. Publish to Tableau Public — File → Save to Tableau Public As…

  3. Copy embed code from the published workbook's Share button

  4. Paste code into this page's tableau.html inside the .tableau-embed div
