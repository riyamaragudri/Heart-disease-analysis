About the Project

Heart Disease Analysis System

HeartWatch Analytics is a full-stack data science project that transforms raw medical records into interactive clinical insights. Built with Python, Flask, and Chart.js — with a Tableau integration guide included.

The project spans data generation, statistical analysis, a REST API backend, and a polished frontend dashboard — making it production-ready and easily extensible.
